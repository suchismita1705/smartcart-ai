# SmartCart AI 🚀

SmartCart AI is a fully functional, dynamic e-commerce web application built with Python and Flask. It features a complete shopping experience, from browsing products to secure authentication, cart management, and a custom built-in product matching chatbot.

## ✨ Features

* **User Authentication:** Secure login and registration system.
* **Smart Chatbot:** A lightning-fast, built-in product matching assistant that recommends real products from the store based on user keywords (No external API required!).
* **Shopping Cart & Checkout:** Add items to the cart, adjust quantities, and process checkouts with unique, auto-generated Order IDs.
* **Wishlist System:** Save favorite products for later.
* **Order Management:** Users can view their past orders and current order status.
* **Automated Emails:** Sends automated order confirmation emails to users upon successful checkout.
* **Dynamic Database:** Powered by SQLite and SQLAlchemy to seamlessly manage users, orders, and wishlist data.

## 🛠️ Tech Stack

* **Backend:** Python, Flask, Flask-SQLAlchemy
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite
* **APIs:** DummyJSON (for initial product catalog seeding)
